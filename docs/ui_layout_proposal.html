以下页面控件逻辑合理，不做改动：
- 河流：3步引导清晰，已优化过默认选择和验证
- 国家：创建→列表→属性 流程OK
- 大洲：列表+拾取 简单够用
- 后勤：邻接+铁路+补给 结构合理
- 密度图/颜色图/默认地图：控件少，不需要调整
- 新大陆：画笔+生成+清除，够简单
总结：改 6 个页面，动 4 类操作

1. 位置调整：一键初始化提到顶部、生成参数和按钮合并
2. 去重复：高度图/地形的重复生成按钮
3. 合并：两个参考图区域合一、工具和画笔大小合一
4. 精简：省份信息改单行、地块类型改横排、按钮并排

不改的：样式/颜色/字号/间距 — 纯逻辑调整，不动 styles.py
